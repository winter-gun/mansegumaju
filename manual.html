만세구마쥬키우기 설명서

마음이 조마조마할 때, 고구마와 함께 숨을 고르고
자유롭게 낙서하는 작은 쉼터예요.

처음 한 번만 해두세요

  1. 받은 링크를 Safari(또는 크롬)에서 열어요.
  2. 우리만 아는 암호를 입력해요. 이 기기에서 처음 한 번만 물어보고, 다음부터는 바로 열려요.
  3. 아이패드라면 공유 버튼(⬆︎) → "홈 화면에 추가"를 눌러두세요. 다음부터는 아이콘만 누르면 앱처럼 전체화면으로 열려요.
  4. 끝! 따로 설치하거나 결제할 것은 아무것도 없어요.

화면은 이렇게 흘러가요

앱을 열면 아롱이가 먼저 인사해요("잠깐 쉬어가자"). 잠시 후 저절로 메인 화면으로 넘어가고, 화면 가운데에서 고구마가 천천히 커졌다 작아졌다 하며 숨을 쉬어요. 화면 전체가 스케치북이라 어디든 바로 그리면 됩니다. 시간 제한도, 정해진 순서도 없어요 — 원하는 만큼 머물다 가면 돼요.

고구마 따라 숨 고르기

고구마가 커질 때 코로 천천히 들이쉬고, 잠시 멈췄다가, 작아질 때 입으로 길게 내쉬어요. 고구마 아래 글씨가 알려주니 그냥 따라가면 돼요.

들이쉬기 4초
멈춤 2초
내쉬기 6초

내쉬는 숨이 더 긴 이유: 길게 내쉴 때 몸의 긴장이 풀리는 스위치가 켜져요. 서너 번만 반복해도 달라지는 게 느껴질 거예요.

그리기 도구

색 고르기

파스텔 다섯 가지 — 장미·풀·하늘·살구·코코아. 동그라미를 누르면 바뀌어요.

펜

기본 도구예요. Apple Pencil로 그리면 누르는 세기에 따라 선 굵기가 살아나요. 손가락이나 마우스도 물론 돼요.

마커

은은하게 비치는 굵은 펜이에요. 같은 곳을 여러 번 칠하면 조금씩 진해져서, 넓게 칠하거나 마음을 문지르듯 쓸 때 좋아요.

수채 브러시

물감이 종이에 스미듯 연하고 부드럽게 번지는 붓이에요. 같은 자리를 천천히 겹칠수록 은은하게 진해져서, 마음을 물들이듯 칠할 때 좋아요.

아롱이 발자국

누르면 발도장이 콩, 찍혀요. 손을 떼지 않고 끌면 발자국이 총총총 이어져요 — 아롱이가 산책하는 것처럼요.

지우개

문지른 자리만 지워져요. 다시 색 동그라미를 고르면 자동으로 그리기로 돌아와요.

선 굵기

누를 때마다 가늘게 → 보통 → 굵게 순서로 바뀌어요. 펜·마커·발자국·지우개 모두에 적용돼요.

되돌리기

방금 그은 선 하나를 취소해요. 여러 번 눌러 계속 되돌릴 수 있어요.

다시 보기

▷ 버튼을 누르면 내 그림이 처음부터 천천히 다시 그려져요. 가만히 바라보기만 하면 돼요. 도중에 화면을 탭하면 바로 원래대로 돌아와요.

아롱이 앨범

하트 버튼을 누르면 아롱이 사진을 한 장씩 넘겨볼 수 있어요 (옆으로 밀거나 화살표). "사진 추가"로 내 사진 보관함의 사진을 직접 넣을 수도 있어요 — 넣은 사진은 인사 화면과 "잘했어요" 화면에도 함께 등장해요. 직접 넣은 사진은 내 기기 안에만 저장돼요.

보관함

저장하기를 누를 때마다 그림이 여기에도 차곡차곡 쌓여요. "다시 보기"는 그 그림을 그렸던 순서 그대로 천천히 재생해주고, "이어 그리기"는 그 그림을 캔버스로 불러와 계속 그릴 수 있게 해줘요. 불러오기 전에 그리던 그림은 자동으로 보관함에 먼저 담아줘서 사라질 걱정이 없고, 수정해서 저장하면 원본은 그대로 두고 새 그림으로 쌓여요. 보관함은 내 기기 안에만 있어요.

백업 — 보관함 아래 "내보내기"를 누르면 그림(직접 추가한 사진 포함)이 파일 하나로 저장돼요. 아이패드에서는 공유 창이 바로 떠서 카톡 "나에게 보내기"로 한 번에 보낼 수 있어요. 다른 기기에서 "가져오기"로 그 파일을 선택하면 그대로 복원됩니다. 보관함을 열면 마지막 백업이 며칠 전이었는지도 조용히 알려줘요.

전체
전체 지우기

화면 전체를 비워요. 실수로 눌러도 괜찮게 두 번 눌러야 지워져요 — 한 번 누르면 "한 번 더 누르면 지워져요"라고 먼저 물어봐요.

저장
저장하기

내 낙서와 고구마가 한 장의 그림(PNG)이 돼요. 매번 다른 아롱이 사진과 다른 인사가 맞아줘요.

저장한 그림 간직하기

  1. 이미지 저장 버튼을 누르면 파일로 내려받아요.
  2. 아이패드에서는 미리보기 그림을 길게 꾹 누르면 "사진에 저장"을 고를 수도 있어요.
  3. 공유하기 버튼이 보이면 메시지·카톡 등으로 바로 보낼 수 있어요.

안심해도 되는 것들

  • 그리던 그림은 자동으로 두 군데에 저장돼요. 실수로 창을 닫거나 새로고침해도, 한쪽 저장이 잘못돼도 다시 열면 그대로 있어요.
  • 선이 저절로 사라지거나, 시간이 다 됐다고 끊는 일은 절대 없어요.
  • 소리·진동·알림이 전혀 없어요.
  • 한 번 열어둔 뒤에는 인터넷이 없어도 열려요 (홈 화면에 추가해두면 가장 확실해요).
  • 앱을 보는 동안에는 화면이 저절로 꺼지지 않아요.
  • 전부 무료이고, 광고나 결제가 생길 일도 없어요.
  • 내 그림은 저장 버튼을 누르기 전까지 내 기기 밖으로 나가지 않아요.
  • 아롱이 사진과 문구는 암호로 잠겨 있어서, 링크를 우연히 알게 된 사람은 열어볼 수 없어요.
  • 앱을 연 시각·횟수는 만든 사람에게 보여요. 그림·이름·위치 같은 건 절대 보이지 않아요.

뭔가 이상할 때

  • 예전 화면(색 4개)이 보여요 — 브라우저를 새로고침해 주세요. 그래도 그대로면 탭을 완전히 닫고 다시 열면 돼요.
  • 펜슬 굵기가 안 변해요 — Apple Pencil 배터리를 확인하고, 화면을 다시 터치해 보세요.
  • 그리던 그림이 안 보여요 — 자동 저장은 같은 기기의 같은 브라우저에서만 이어져요. 늘 쓰던 곳(홈 화면 아이콘)으로 열어 주세요.
  • 페이지가 안 열려요 — 본인 Claude 계정으로 로그인돼 있는지 확인해 주세요.
